Dreamweaver Templates (HTML5Boilerplate and Twitter Bootstrap)

Feel free to fork this and start with a basic template or if you have markup to start with, use that. Its really simple and uses single dreamweaver template assigned to index.html file. You can copy and rename template to start with your own and assign it to pages. If you are unfamiliar with dreamweaver templates I suggest you read or view some tutorials on these at Adobe or any other sites.
